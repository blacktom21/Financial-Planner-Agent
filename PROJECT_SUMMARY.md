# Financial Advisor AI - Project Summary

## Executive Summary

Financial Advisor AI is a comprehensive, intelligent financial planning system that leverages multi-agent architecture and LLM technology to provide personalized financial advice. The system helps users manage expenses, plan investments, and achieve financial goals through an intuitive, beginner-friendly interface.

## Technology Stack

- **Backend**: Python 3.11+, Flask 3.1.2
- **Database**: SQLite (development), PostgreSQL-ready
- **LLM**: Hugging Face API / Ollama / Rule-based fallback
- **Frontend**: HTML5, CSS3, JavaScript (Vanilla), Chart.js
- **Authentication**: Flask Sessions, Werkzeug Security
